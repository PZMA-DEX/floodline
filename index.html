침수 위험 알림 신청

주소와 단계를 선택하면 해당 단계 이상 진입 시 즉시 알림을 보내드립니다.

실제 발송은 백엔드 연결 후 활성화 · 개인정보 수집 최소화

알림이 작동하는 방식

01

강우 감지 → 정적 취약도 결합

기상청 AWS 5km 격자 강우강도를 10분마다 수신, 건물 블록 취약도와 실시간 결합합니다.

02

위험 단계 판정

합리식 기반 Q유입/Q설계 비율 + 반지하·저지대 가중치로 4단계(안전·주의·경계·위험) 판정.

03

FCM / 카카오 알림톡 발송

선택 단계 진입 시 해당 블록 구독자에게 푸시. "이 건물은 침수 직접 피해 위험 구역".

04

10~20분 먼저 알림

초단기예보 +1h 예측을 함께 활용해 광역 알림보다 10~20분 선행 경보를 목표로 합니다.
